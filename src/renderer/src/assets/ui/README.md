# UI chrome

Decorative UI artwork lives here.

- `compass-dial.png` — 188×188 year-dial background preview asset.
- `compass-dial@2x.png` — **required raster asset.** The year-dial background overlay.
  **No text / no numerals** (the app draws the year + era on top, ⌀120 centre kept clear),
  transparent outside the circle.
- `compass-dial@3x.png` — high-density raster asset.
- `compass-needle.svg` — *optional.* Centred needle pointing up, for a future "spinning" dial.
- `brand-mark.svg` — *optional.* 40×40 `currentColor` compass glyph for the header logo.

Full spec: [`/DESIGN_ASSETS.md`](../../../../../DESIGN_ASSETS.md).
